Gravity and Orbits

This project explores the physics of gravity and orbits using a model of the sun and Earth. This project is my favorite because it is accurate, it is interactive because you can change the code, and it shows how orbits work.

Stature Project

This project shows you the mean and standard deviation of a list. You can change the list if you want to see different results. I like this project because it shows not just the heights and mean, but also the standard deviation. I also like how it is interactive.

View Project on Scratch

Random Walk

This project is a random walk with 8 different directions, 4 straight and 4 diagonal. I like it because it creates a new pattern every time and it is satisfying to watch.
